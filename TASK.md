# Задание
## Необходимо реализовать графическое приложение (на C++), которое будет отображать дерево файловой системы

- Взять за основу проект dirview
- Сделать стартовой директорией домашнюю директорию текущего пользователя. То есть того пользователя что запускает утилиту
- Отображать файлы, папки, в том числе и скрытые
- Добавить QLineEdit виджет, который использовать для фильтрации по имёнам файлов и папок
- Желательно: добавить столбец "Размер папки" или переопределить текущий ("Размер"), где для каждой папки будет отображена кнопка «Обновить», по нажатию на которую необходимо определить и отобразить размер папки в этой же ячейке, рядом или вместо кнопки.
- Желательно+: дебианизировать готовое решение (dpkg-buildpackage)
- Желательно+: Изменения относительно исходного кода (itemviews/dirview) оформить в виде .patch файла (debian/patches/.patch)
- Желательно+: Настроить ci/cd в проекте

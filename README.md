# Dir Tree Viewer

Графическое приложение на Qt для просмотра дерева файловой системы с поддержкой фильтрации, сортировки и поиска.

## Описание

Приложение отображает структуру файловой системы в виде дерева и позволяет выполнять поиск файлов и каталогов по имени. 
По умолчанию в качестве стартовой директории используется домашний каталог текущего пользователя. 

Проект разработан на основе [dirview](https://doc.qt.io/archives/qt-5.15/qtwidgets-itemviews-dirview-example.html).

## Возможности
### Фильтрация

Пользователь вводит в строку поиска регулярное выражение, по которому ищутся файлы и директории. В обычном режиме поиск идет только в корневой директории дерева. 
В режиме глубокого поиска рекурсивно обходятся все подкаталоги. В дереве отображаются найденные элементы, а также директории,
необходимые для сохранения пути к найденным элементам.

<img width="808" height="634" alt="image" src="https://github.com/user-attachments/assets/d3ebef48-1d0b-4597-aa8e-7e5511a9c68d" />

Также доступна функция учета регистра.

### Сортировка

При нажатии на заголовок столбца выполняется сортировка элементов дерева
по соответствующему столбцу в порядке возрастания или убывания.

### Размер директорий

Для каталогов отображается надпись "Обновить", при двойном щелчке на которую выполняется рекурсивный подсчёт размера каталога.
Полученный результат сохраняется в кэше и отображается в таблице.

## Компоненты приложения

### DirTreeModel

Наследует `QFileSystemModel`. Представляет собой модель системы файлов - предоставляет информацию о файлах и директориях файловой системы. 

Отвечает за работу с файловой системой и хранение кэша рассчитанных размеров. 

Поддерживает ленивую загрузку по мере обращения к узлам дерева.

### DirFilterModel

Наследует `QSortFilterProxyModel`. Является прокси-посредником между моделью и отображением. 
Применяет фильтрацию файлов по регулярному выражению. 

Узел проходит фильтр, если выполняется хотя бы одно из условий:
- имя узла содержит паттерн поиска.
- один из потомков узла содержит паттерн (в режиме глубокого поиска).

## DirTreeView

Наследует QTreeView. Отвечает за визуальное отображение дерева. Принимает DirFilterModel как модель.

### MainWindow

Главное окно приложения.

Помимо виджета дерева файлов содержит:
- Строку поиска;
- Кнопку «Учитывать регистр»;
- Кнопку «Глубокий поиск».

## Сборка

### Требования

- C++17+;
- CMake;
- Qt 5.15.

### Сборка с CMake

```bash
cmake -S . -B build
cmake --build build
```

### Запуск

```bash
./dir_view_test [опции] [директория]
```

### Параметры
```text
  -h, --help     Вывод справки по параметрам командной строки.
  --help-all     Вывод справки с особыми параметрами Qt.
  -v, --version  Показать сведения о версии.
  -c             Отключить отображение пользовательских иконок.
  -w             Отключить отслеживание изменений.
```
### Аргументы
`директория` - начальная директория дерева (абсолютный путь)

### Сборка пакета deb

Для проекта подготовлена конфигурация Debian-пакета в каталоге `debian/`, включающая информацию о приложении.

Изменения относительно исходного проекта dirview оформлены в виде патча `dir-view-test.patch` в `debian/patches/`.

Сборка выполняется командой:

```bash
dpkg-buildpackage -us -uc
```

После успешной сборки в родительском каталоге будут созданы файлы:

```bash
../dir-view-test_1.0-1_amd64.deb
../dir-view-test_1.0-1_amd64.buildinfo
../dir-view-test_1.0-1_amd64.changes
```

Установить пакет можно командой:

```bash
sudo dpkg -i ../dir-view-test_1.0-1_amd64.deb
```
